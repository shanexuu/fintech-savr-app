## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/shanexuu/fintech-savr-app.git
   ```

2. Navigate to the project directory

   ```bash
   cd savr-app

   ```

3. Install dependencies

   ```bash
   npm install
   ```

4. Start the app

   ```bash
    npx expo start
   ```

5. Open the app using Expo Go on your device or an emulator.
